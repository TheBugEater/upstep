# Upstep backlink campaign

Goal: earn relevant referring domains and qualified product traffic without paid-link or bulk-directory tactics.

## Positioning

**Name:** Upstep

**URL:** https://upstep.dev

**Tagline:** Feedback and feature voting inside your app.

**Short description:** Upstep helps web and mobile teams collect feedback, let users vote on ideas, manage requests on flexible boards, and connect the backlog to AI coding agents through MCP.

**Long description:** Upstep is a feedback and feature-voting platform for web and mobile applications. Teams can embed a native feedback experience with JavaScript, React Native, or Flutter, collect ideas and bug reports without sending users to another website, prioritize requests by votes, publish a roadmap, and let AI coding agents triage feedback through a project-scoped MCP server. A free plan is available without a credit card.

**Categories:** Customer feedback; product management; feature voting; public roadmap; developer tools; MCP server; SaaS.

**Pricing:** Free plan available; paid plans for growing teams.

**Primary differentiators:**

- Embedded web and mobile feedback collection
- JavaScript, React Native, and Flutter SDKs
- Feature voting with anonymous-user deduplication
- Flexible feedback boards and public roadmaps
- Project-scoped MCP integration for coding agents

## Operating cadence

Each week:

1. Submit to three qualified directories from `prospects.csv`.
2. Send five individually checked resource-page pitches.
3. Follow up once after seven days, then close the opportunity.
4. Record the live URL and whether the link is followed, nofollowed, or absent.
5. Track referral visits and signups alongside referring domains.

Do not pay for followed links, automate submissions, use keyword-stuffed anchor text, or request links from unrelated sites.
