# @upstep/types

Shared TypeScript types for the [Upstep](https://upstep.dev) feedback SDKs: `UpstepConfig`, `Feedback`, `Comment`, and the request/response shapes used by [`@upstep/js`](https://www.npmjs.com/package/@upstep/js) and [`@upstep/react-native`](https://www.npmjs.com/package/@upstep/react-native). Flutter support lives in the separate [`upstep_flutter`](https://pub.dev/packages/upstep_flutter) package.

You don't need to install this directly — it comes along as a dependency of either SDK above. It's published standalone so both SDKs (and anyone calling the Upstep API directly from TypeScript) share one source of truth for the types.
